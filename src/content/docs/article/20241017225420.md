# Permanent Notes

Zettelkastenの本体

- 他人に公開できるレベルの永続的に残すメモ
- [[20241017225231|Fleeting Notes]]と[[20241017225302|Literature Notes]]を統合し清書したもの
- Structure Notesも合わせて管理
  - 複数のPermanent Notesを集約したもの
  - 構造を表現できる
- Index Notesはタグ機能で実現
